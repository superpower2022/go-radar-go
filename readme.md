## 2.文件目录

- configs : deep_sort的配置文件

- data：测试视频与图片

- deep_sort：框架

  -----------deep： 特征提取（cnn网络）

  ​        |---sort  ： 具体算法

- models：yolov5的模型

- pnp ： 测距

- tools：其他杂项功能函数

- yolov5：物体识别框架

- docs：说明文档

- demo.py -------------- python3 demo.py 直接运行！

- demo2.py ------------- 对于demo.py的更新，能在雷达站主机上直接运行

- communicator：与A板通信的代码

- detect.py ------------ 使用训练好的神经网络用于检测车辆位置

- Camera.py ------------ 使用Zed深度相机获取实时图像，综合了神经网络与A板通讯模块，完成雷达站整体流程

- main.py -------------- 直接运行，开启雷达站
